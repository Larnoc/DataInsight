# DataInsight
Проект DataInsight: Руководство по аналитике в Loginom

Цель проекта: Разработка практического руководства по использованию аналитической платформы Loginom на примере сквозного бизнес-кейса.

Основные задачи:
1) Интеграция данных: Подключение к различным источникам и консолидация информации.
2) Очистка и трансформация: Обработка пропусков, дедупликация и нормализация данных с помощью встроенных узлов.
3) Аналитическое моделирование: Проектирование логических схем обработки и применение алгоритмов (классификация, кластеризация или прогнозирование).
4) Визуализация: Настройка дашбордов для интерпретации результатов.

Структура репозитория:

/CODE — файлы пакетов Loginom (.lgp) с настроенными сценариями.
/DOCS — подробная документация, инструкции и пояснительные записки к лабораторным работам.
